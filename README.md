# ✂️ CutOnly - 動画カット検出ツール

`CutOnly`は、動画ファイルからシーンの切り替わり（カット）を自動で検出するためのシンプルなWebアプリケーションです。
`cidcutV2`のカット検出機能をベースにしており、より手軽に利用できるようにUIを改善しました。

## ✨ 主な機能

- **簡単なファイルアップロード**: ドラッグ＆ドロップで動画ファイルをアップロードできます。
- **インタラクティブな設定**: 3つの検出方法（Content, Adaptive, Threshold）と、検出する最小の長さを画面上で簡単に設定できます。
- **リアルタイム進捗表示**: 解析中の進捗がプログレスバーで表示され、処理状況を把握できます。
- **結果の確認とダウンロード**: 検出されたカットの一覧（開始フレームと終了フレーム）を画面で確認し、結果をJSON形式でダウンロードできます。

## 🚀 使い方

1.  **リポジトリのクローンと移動**
    ```bash
    git clone <リポジトリのURL>
    cd cutonly
    ```

2.  **仮想環境の作成と有効化**
    ```bash
    # Windows
    python -m venv .venv
    .venv\Scripts\activate
    
    # macOS / Linux
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3.  **必要なライブラリのインストール**
    ```bash
    pip install -r requirements.txt
    ```

4.  **アプリケーションの起動**
    ```bash
    streamlit run cut_detector.py
    ```

5.  **ブラウザで操作**
    -   起動後に表示されるURL（通常は `http://localhost:8501`）をブラウザで開きます。
    -   画面の指示に従って動画ファイルをアップロードし、設定を選択して解析を開始してください。

## 🛠️ 検出アルゴリズムについて

-   **Content**: フレーム間の色の変化を比較して、シーンの切り替わりを検出します。最も一般的で、多くの場合にうまく機能します。
-   **Adaptive**: 明るさの急激な変化に基づいてカットを検出します。フェードイン・フェードアウトのような場面に強いです。
-   **Threshold**: 全てのピクセルの色の変化が、設定した閾値を超えた場合にカットとして検出する、最もシンプルな方法です。
